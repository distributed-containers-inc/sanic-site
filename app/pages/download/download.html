Download

Sanic is currently only officially supported for GNU/Linux x64

Download the latest release!

After downloading, take a look at the Example project, as well as the Interactive tutorial and quick start guide.

You'll also need valid Docker and Kubectl binaries on your path.

All there is to do is add sanic to your PATH, i.e., sudo cp ~/Downloads/sanic /usr/local/bin/

After adding sanic to your path, there is no other installation required. The first command you'll likely run is "sanic deploy" to start the kubernetes cluster in your local docker, please have patience because this step only ever needs to be slow once!

Other platforms

Try downloading the source code and building it for your own platform!
