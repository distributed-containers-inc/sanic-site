Download

Sanic is currently only officially supported for GNU/Linux x64

Download the latest release!

After downloading, take a look at the Example project, as well as the Interactive tutorial and quick start guide.

You'll also need valid Docker and Kubectl binaries on your path, with docker set up to not require sudo (i.e., docker ps should work without sudo)

All there is to do is:

  1. Install docker and kubectl
  2. Allow access to docker, i.e., sudo usermod -a -G docker $(whoami) and logging out / back in
  3. Add sanic to your PATH, i.e., sudo cp ~/Downloads/sanic /usr/local/bin/
  4. Make sanic executable, i.e., sudo chmod 755 /usr/local/bin/sanic

After adding sanic to your path, there is no other installation required.

Other platforms

Try downloading the source code and building it for your own platform!
